## 3. redux-saga [\*](https://redux-saga.js.org)

```bash
npm i redux-saga
```

- 미들웨어 입니다.
- 제너레이터 객체를 만들어 내는 제네레이터 생성 함수를 이용합니다.
- 리덕스 사가 미들웨어를 설정하고, 내가 만든 사가 함수를 등록한 후, 사가 미들웨어를 실행합니다.
- 그리고 등록된 사가 함수를 실행할 액션을 디스패치하면 됩니다.
